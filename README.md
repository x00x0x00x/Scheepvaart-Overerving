# Scheepvaart-Overerving
Het managen van een Rederij door middel van Vloten, Havens en Trajecten. De verschillende soorten Schepen (en hun properties / methods) bevinden zich op diverse niveaus van overerving. 
De Overzicht klasse om zicht te krijgen op de applicatie (hoofdzakelijk mbt klasse Rederij).

- Refactoring
-- Diverse stukken code mbt de overzichtsklasse kunnen verplaatst worden naar de relevante klasses per methode om de hoeveelheid code in de overzichtsklasse zo miniem mogelijk te houden en het creeeren van een "God class" te vermijden.
